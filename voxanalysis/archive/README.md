# Archive

Historical material kept for reference. **Nothing here is live** — none of it
is imported, executed, or read by the analysis engine, the downloader, or the
coaching pipeline. It is safe to ignore during normal work, and it was moved
(not deleted) so the git history and the content are both preserved.

```text
archive/
├── handoffs/          one-time agent handoff docs (completed work) + old root HANDOFF.md/HANDOFF-HOWARD.md
├── docs/              superseded design/process notes (saas-mvp, phase1-flow, training-plan-expansion,
│                      pdf-reporting, and the early implementation write-ups)
└── scratch-analyses/  ad-hoc analysis JSONs produced while developing (individual takes and one-off
                       reference runs) — distinct from the curated calibration pack in
                       vox-analysis/engine/calibration/references/
```

Current, authoritative documentation lives in `../docs/` and in each part's own
`README.md`. If something here contradicts those, the current docs win.
